    ANGULAR JS

    Définition : Framework JavaScript libre et open-source développé par Google. Angular permet, grâce à ses balises, d’utiliser une sorte d’html étendu et d’implémenter le patron MVC.
    Tutoriels :

    • http://blog.zenika.com/2013/05/22/angularjs-getting-started/
    • http://toddmotto.com/opinionated-angular-js-styleguide-for-teams/
    • https://github.com/toddmotto/angularjs-styleguide
    • https://github.com/a8m/angular-filter
    Notions importantes :
    • Contrôleur : attaché à la vue par la directive ng-controller, il permet de gérer une zone géographique de la page web.
    • Directive : outil permettant d’étendre les capacités du langage HTML. Les directives fournies avec le framework se présentent sous la forme ng-something
    • Service : outil permettant de partager du code dans toute l’application grâce à l’injection de dépendance. Exemple : $scope, $http, $timeout
      • $http :
        • data : retourne les données récupérées dans le fichier appelé.
        • status : retourne le code status de l'appel (200 à 299 : success, autre : error)
        • headers : retourne les informations sur l'en-tête récupéré.
        • config : retourne un objet comprennant l'ensemble des informations de configuration de la demande.
    • Promises :
    • Factory :
    Installation : fichier angular.js

    CORDOVA

    Définition : Framework open-source permettant de développer des applications hybrides (Web, Android, iOS, Windows Phone) grâce au langages HTML, CSS et JavaScript. Cordova était anciennement appelé Phonegap.
    Installation / Commandes importantes :
    > npm install –g cordova
    > Cordova create hello com.example.hello HelloWorld
    > Cordova platform add android --save
    > cordova build

    SASS 

    Définition : Langage informatique de génération dynamique de feuilles de style. SassScript est un langage de script interprété en CSS, la nouvelle syntaxe, SCSS a un formalisme proche de CSS.

    LESS (http://lesscss.org/ ) : Définition : Langage informatique de génération dynamique de feuilles de style. Il a d’abord été écrit en Ruby mais les versions récentes sont en JavaScript. LESS peut être compilé à la volée par le navigateur ou par le serveur ou bien pré-compilé en un fichier CSS. LESS permet d’ajouter au CSS, entre autres, la gestion des variables et l’héritage. Notions importantes : Variables : On peut définir des variables afin de gagner en lisibilité et flexibilité.
