    JUNIT

    Définition : Tests unitaires

    JASMINE

    Définition : Framework permettant de faire des tests unitaires pour du code JavaScript
    Installation : Npm install –g jasmine-core
    Npm install -g jasmine –save-dev
     Notions importantes :

    • Runner : outil responsable de l’exécution des tests. Certains permettent de ne lancer qu’une seule catégorie de tests
    • Reporter : affiche les résultats d’une série de tests Les tests sont écrits dans le fichier de spécification

    KARMA

    Définition : Tests unitaires
    Tutoriels : http://atomrace.com/tester-son-application-angular-avec-karma-et-jasmine/
    Installation : Npm install –g karma
    Npm install karma –save-dev
    Npm install –g karma-cli
    Npm install –g generator-karma
    npm install karma-firefox-extra-launcher --save-dev
    npm install karma-ie -launcher --save-dev
    npm install –g karma-jasmine –save-dev
    Karma init
    Karma start –log-level debug

    MAVEN

    Définition : Outil de gestion et d’automatisation pour JAVA SE et JAVA EE. Maven permet de produire un logiciel à partir de ses sources en optimisant les tâches nécessaires.
    Il est recommandé d'utiliser Maven en ligne de commande autant que possible ! L'intégration avec les IDE n'est pas toujours très heureuse...!
    Notions importantes : POM.xml : POM signifie Project Object Model, c’est le fichier décrivant le projet, ses dépendances, ses modules ainsi que l’ordre à suivre pour sa production

    NODE.JS

    Définition : Plate-forme logicielle (environnement permettant la gestion et/ou l’utilisation de services applicatifs) libre en JavaScript orientée vers les applications réseau.
    Node.JS contient notamment un serveur http intégré permettant de faire tourner un serveur web sans Apache.
