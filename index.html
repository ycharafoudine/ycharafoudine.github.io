Glossaire JAVASCRIPT Définition : Langage de programmation de scripts principalement employé dans les pages web interactives Tutoriels : http://lodash.com/docs https://developer.mozilla.org/fr/docs/Web/JavaScript/Une_r%C3%A9introduction_%C3%A0_JavaScript http://momentjs.com/docs/ Installation : ANGULAR JS (https://angularjs.org/) : Définition : Framework JavaScript libre et open-source développé par Google. Angular permet, grâce à ses balises, d’utiliser une sorte d’html étendu et d’implémenter le patron MVC. Tutoriels : http://blog.zenika.com/2013/05/22/angularjs-getting-started/ http://toddmotto.com/opinionated-angular-js-styleguide-for-teams/ https://github.com/toddmotto/angularjs-styleguide https://github.com/a8m/angular-filter Notions importantes : Contrôleur : attaché à la vue par la directive ng-controller, il permet de gérer une zone géographique de la page web. Directive : outil permettant d’étendre les capacités du langage HTML. Les directives fournies avec le framework se présentent sous la forme ng-something Service : outil permettant de partager du code dans toute l’application grâce à l’injection de dépendance. Exemple : $scope, $http, $timeout Promises : Factory : $http : data : retourne les données récupérées dans le fichier appelé. status : retourne le code status de l'appel (200 à 299 : success, autre : error) headers : retourne les informations sur l'en-tête récupéré. config : retourne un objet comprennant l'ensemble des informations de configuration de la demande. Installation : fichier angular.js JQUERY (http://jquery.com/) : Définition : Bibliothèque JavaScript libre et multi-plateforme créée pour faciliter l’écriture de scripts dans le code HTML des pages Web Tutoriels : http://geekos.fr/02/2012/jquery-astuces-et-conseils-pour-ameliorer-les-performances/ http://learn.jquery.com/performance/ http://lab.abhinayrathore.com/jquery-standards/ Notions importantes CORDOVA (https://cordova.apache.org/docs/en/latest/) : Définition : Framework open-source permettant de développer des applications hybrides (Web, Android, iOS, Windows Phone) grâce au langages HTML, CSS et JavaScript. Cordova était anciennement appelé Phonegap. Installation / Commandes importantes : npm install –g cordova Cordova create hello com.example.hello HelloWorld Cordova platform add android --save cordova build IONIC (http://ionicframework.com/) : Définition : Framework pour développement hybride (Web, Android, iOS, Windows Phone) Installation : npm install –g ionic CSS3 : Définition : Langage informatique décrivant la présentation des feuilles HTML et XML. SASS (http://sass-lang.com/) : Définition : Langage informatique de génération dynamique de feuilles de style. SassScript est un langage de script interprété en CSS, la nouvelle syntaxe, SCSS a un formalisme proche de CSS. LESS (http://lesscss.org/ ) : Définition : Langage informatique de génération dynamique de feuilles de style. Il a d’abord été écrit en Ruby mais les versions récentes sont en JavaScript. LESS peut être compilé à la volée par le navigateur ou par le serveur ou bien pré-compilé en un fichier CSS. LESS permet d’ajouter au CSS, entre autres, la gestion des variables et l’héritage. Notions importantes : Variables : On peut définir des variables afin de gagner en lisibilité et flexibilité. Exemple : @color: #4D926F BOOSTRAP (http://getbootstrap.com/ ) : Définition : Framework dédié à la création de sites et d’applications web. Son utilisation permet d’avoir facilement une apparence uniforme et responsive sur tout le site. BOWER (https://bower.io/ ) : Définition : Gestionnaire de paquets optimisé pour le front-end. Installation : npm install –g bower (nécessite Node.JS, npm et Git) GRUNTJS (http://www.siteofficiel.com) : Définition : Automatiseur de tâches (task-runner) GULP (http://www.siteofficiel.com) : Définition : Automatiseur de tâches (task-runner) SVN (http://www.siteofficiel.com) : Définition : Logiciel de versionnage GIT (http://git-scm.com/ ) : Définition : Logiciel de versionnage. Certains permettent de ne lancer qu’une seule catégorie de tests Reporter : affiche les résultats d’une série de tests Les tests sont écrits dans le fichier de spécification KARMA (http://www.siteofficiel.com) : Définition : Tests unitaires Tutoriels : http://atomrace.com/tester-son-application-angular-avec-karma-et-jasmine/ Installation : Npm install –g karma Npm install karma –save-dev Npm install –g karma-cli Npm install –g generator-karma npm install karma-firefox-extra-launcher --save-dev npm install karma-ie -launcher --save-dev npm install –g karma-jasmine –save-dev Karma init Karma start –log-level debug MAVEN (https://maven.apache.org/ ) : Définition : Outil de gestion et d’automatisation pour JAVA SE et JAVA EE. Maven permet de produire un logiciel à partir de ses sources en optimisant les tâches nécessaires.
